###  记录NLP的学习过程，保存学习资料

#### 2017.07.11 到 2017.07.18，通读了基于词表的分词程序friso，这是我学习的第一份分词程序源代码。
* Segmentation\friso-master
* 作者C语言功底深厚，工程代码老到。从中，我学到了很多内存管理、实用底层数据结构的实现方法，受益匪浅。
* friso自行实现了动态数组、双向链表、string buffer类、基于bucket的hash表等底层数据结构，以及基于hash表的字典并支持UTF8和GBK两种字典编码。
* 每一种数据结构都有相应的测试代码，可以用make debug编译再gdb，也是读懂程序的入口。
* fmm、mmseg的代码逻辑十分清晰、易读。但英-中、中-英混合、数字混合后再切分，这几种模式的代码读起来比较费力，没有完全理解，以后有需要再看。

#### 2017.07.21 到 2017.07.24，通读、单步了“200行Python代码实现感知机词性标注器”，这是我学习的第一份词性标注源代码。
LiHang\Ch02\AveragedPerceptronPython
